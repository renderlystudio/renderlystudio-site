# NombreEstudio — Sitio web (GitHub Pages)

Plantilla minimalista para catálogo de assets 3D.

Cómo usar:
1. Crea un repositorio en GitHub y sube los archivos `index.html`, `styles.css`, `script.js` y `README.md`.
2. Activa GitHub Pages desde la rama `main` (Settings > Pages) o usando la rama `gh-pages`.
3. Personaliza imágenes y metadatos; puedes mantener los `download` como enlaces temporales hasta integrar una tienda.
4. Para dominio personalizado: añade un archivo `CNAME` con tu dominio en la raíz del repositorio y configura los DNS (A/ALIAS o CNAME según GitHub Pages).
